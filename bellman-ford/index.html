Pseudocode

BellmanFord(G, s):
for each v: d[v] = INF
d[s] = 0
for k = 1 to n-1:
for each edge (u,v,w) in E:
if d[u]+w < d[v]:
d[v] = d[u]+w
parent[v] = u
// Negative cycle check
for each edge (u,v,w) in E:
if d[u]+w < d[v]: CYCLE!
return d[], parent[]
